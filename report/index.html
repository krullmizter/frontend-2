Reflektion och feedback

Tyckte att detta projekt var intressant och roligast av alla våra projekt hittils. Jag lämnade in projektet en vecka för sent men det gjorde jag för att läsa mer om JavaScript och få gjort så mycket som jag kunde.

Tyckte att deluppgif: #1 - VPN, #5 - Paint och #6 - Rock, paper, scissors var lättast att göra, jag spenderade minst tid på dom tre. Att få all info till #1 var inte svårt, kanske bara det med vilken webbläsare man använder. #5 gjorde vi på en lektion och det som lämnade kvar var inte så svårt att Googla. Uppgift #6 var ganska lätt att fundera ut, och efter lite testande med if-satser så funkade det till sist (dock så fungerar inte bytet mellan stora och små namn i sten, sax samnt påse...).

#2 - Memory tog lång tid men det var nog för att jag inte kom nån vart med sjäva flip funktionen, men efter den kom på plats så tog det inte länge att fixa allt annat. Jag skulle vela skapa ett memory spel som slumpmässigt kastar om korten, men nu så är dom bara hårdkodade på plats. #4 - Stopwatch är egentligen inte heller svår, men fick inte if-satserna samt setInterval att fungera riktigt bra. #7 - Pusslet vet jag inte varför det var krånligt, det var nog att jag lämnade det bara för det såg så lätt ut, och sen var den inte helt lätt tillsist. #5 - Gallery är nog något flera programmerare krånglar med, man tror det är lätt att skapa lightboxes och gallery men det strular alltid... intervallerna bråkade länge och den fungerar sådär helt okej nu, lightbox effekten borde förbättras.

Projektet var kul att arbeta med, tidsplaneringen skulle jag ha behövt få in då det lämnade efter med en vecka men nu är jag ändå helt nöjd. Skulle vela förbättra en del saker med JavaScript, men nu är det som det är. Helt nöjd är jag med min prestation och har haft flera "yay" moments då koden fungerar som den ska (samt flera "ney" moments också...).
